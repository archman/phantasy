parser
======

*import* interface:

.. code-block:: python

    from phantasy import parser

Within the namespace of *parser*, the following could be reached:
``Configuration``, ``find_machine_config``.

*func_class* may be imported this way, though it is not recommanded:

.. code-block:: python

    from phantasy import <func_class>

or

.. code-block:: python

    from phantasy.library.parser import <func_class>

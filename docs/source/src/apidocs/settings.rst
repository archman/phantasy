settings
========

*import* interface:

.. code-block:: python

    from phantasy import settings

Within the namespace of *settings*, the following could be reached:
``Settings``, ``build_flame_settings``, ``build_impact_settings``.

*func_class* may be imported this way, though it is not recommanded:

.. code-block:: python

    from phantasy import <func_class>

or

.. code-block:: python

    from phantasy.library.settings import <func_class>

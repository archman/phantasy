physics
=======

*import* interface:

.. code-block:: python

    from phantasy import physics

Within the namespace of *physics*, the following could be reached:
``Distribution``, ``Point``, ``Line``.

*func_class* may be imported this way, though it is not recommanded:

.. code-block:: python

    from phantasy import <func_class>

or

.. code-block:: python

    from phantasy.library.physics import <func_class>

scan
====

*import* interface:

.. code-block:: python

    from phantasy import scan

Within the namespace of *scan*, the following could be reached:
``BaseScanClient``, ``ScanClient1D``, ``ScanDataFactory``.

*func_class* may be imported this way, though it is not recommanded:

.. code-block:: python

    from phantasy import <func_class>

or

.. code-block:: python

    from phantasy.library.scan import <func_class>

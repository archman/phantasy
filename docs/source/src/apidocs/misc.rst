misc
====

*import* interface:

.. code-block:: python

    from phantasy import misc

Within the namespace of *channelfinder*, the following could be reached:
``flatten``, ``get_intersection``, ``machine_setter``, ``bisect_index``,
``pattern_filter``, ``expand_list_to_dict``, ``simplify_data``,
``complicate_data``, ``SpecialDict``, ``parse_dt``.

*func_class* may be imported this way, though it is not recommanded:

.. code-block:: python

    from phantasy import <func_class>

or

.. code-block:: python

    from phantasy.library.misc import <func_class>

layout
======

*import* interface:

.. code-block:: python

    from phantasy import layout

Within the namespace of *layout*, the following could be reached:
``build_layout``, ``Layout``,
``BCMElement``, ``BLElement``, ``BLMElement``, ``BPMElement``,
``BendElement``, ``CavityElement``, ``ColumnElement``,
``CorElement``, ``DriftElement``, ``EBendElement``,
``EMSElement``, ``EQuadElement``, ``ElectrodeElement``,
``FCElement``, ``HCorElement``, ``PMElement``,
``PortElement``, ``QuadElement``, ``SeqElement``,
``SextElement``, ``SolCorElement``, ``SolElement``,
``StripElement``, ``VCorElement``, ``VDElement``, ``ValveElement``,
``Element``, ``Fields``.

*func_class* may be imported this way, though it is not recommanded:

.. code-block:: python

    from phantasy import <func_class>

or

.. code-block:: python

    from phantasy.library.layout import <func_class>

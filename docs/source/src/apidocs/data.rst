data
====

*import* interface:

.. code-block:: python

    from phantasy import data

Within the namespace of *data*, the following could be reached:
``FitModel``, ``gaussian_fit``.

*func_class* may be imported this way, though it is not recommanded:

.. code-block:: python

    from phantasy import <func_class>

or

.. code-block:: python

    from phantasy.library.data import <func_class>

operation
=========

*import* interface:

.. code-block:: python

    from phantasy import operation

Within the namespace of *operation*, the following could be reached:
``create_lattice``, ``load_lattice``, ``MachinePortal``.

*func_class* may be imported this way, though it is not recommanded:

.. code-block:: python

    from phantasy import <func_class>

or

.. code-block:: python

    from phantasy.library.operation import <func_class>
